## üöÄ Project Overview

This project is a sophisticated test automation solution that navigates to the *El Pa√≠s* Opinion section, scrapes article data, translates headers using a third-party API, and analyzes sentiment/word frequency‚Äîall while validating visual integrity across **5 different browser combinations** simultaneously.

Built with **Java** and **Selenium**, it leverages the **BrowserStack Automate** platform to ensure compatibility across real mobile devices and desktop environments.

### üéØ Key Capabilities
*   **Scalable Architecture**: Uses the **Page Object Model (POM)** for maintainability and separation of concerns.
*   **Resilient Automation**: Implements smart waits, retry mechanisms, and robust error handling to prevent flakiness.
*   **Cloud Integration**: optimized for BrowserStack with session tagging, status reporting, and idle timeout prevention.
*   **API Harmony (Dual Mode)**: 
    *   **Primary**: Supports **Google Translate API** via RapidAPI (requires key).
    *   **Fallback**: Automatically switches to **Google Translate (Free)** endpoint if no key is provided, ensuring zero-configuration execution.
*   **Visual Verification**: Downloads high-res article images to a local directory for visual checks.

## ‚ö° Setup & Execution

### Prerequisites
*   Java JDK 11+
*   Maven 3.6+
*   [BrowserStack Account](https://www.browserstack.com/) (Username & Access Key)
*   [RapidAPI Key](https://rapidapi.com/) (Optional, for translation)

### 1. Clone & Configure
```bash
git clone https://github.com/your-username/elpais-selenium.git
cd elpais-selenium
```

### 2. Run on BrowserStack (Recommended)
Execute the specific Maven profile to trigger the cross-browser suite:

```powershell
mvn test -Pbrowserstack `
    "-Dbrowserstack.username=<YOUR_USER>" `
    "-Dbrowserstack.accessKey=<YOUR_KEY>"
```

### 3. Run Locally (Headless)
For quick debugging without cloud credits:
```powershell
mvn test
mvn clean test -Dtest=ElPaisTest
```
